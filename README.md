## Get started

##### - Install dependencies

```sh
yarn install
```

##### - Enable husky

```sh
yarn husky install
```

```sh
chmod ug+x .husky/*
```

```sh
chmod ug+x .git/hooks/*
```

##### - Start dev server

```sh
yarn dev
```
